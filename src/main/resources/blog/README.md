## Hexo博客使用帮助
1、安装Hexo插件执行命令npm install hexo-cli -g

2、新建一篇博客执行命令hexo new post "article title"即可生成博客文件

3、生成静态文档执行命令hexo generate,简写为 hexo g 

4、部署发布命令hexo deploy,简写为hexo d,部署之前需编辑_config.yml文件配置deploy
